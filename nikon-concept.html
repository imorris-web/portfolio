Description

This unoficial design project revamps Nikon's brand value for modern digital applications. The ultimate goal of this rebrand is to create stronger brand recognition by connecting people with the limitless creative potential of Nikon technology. The new logo maintains the ergonomic feeling of Nikon's products, as well as the bold and forward thinking characteristics of the company.

This project features a redesigned website for Nikon USA, showing the application of the new brand for a specifc target market.
